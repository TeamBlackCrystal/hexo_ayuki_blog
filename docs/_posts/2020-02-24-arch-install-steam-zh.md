## 概要

sudo pacman -S steam

しかしながら、ArchLinux では 32bit パッケージがデフォルトでは無効化されている為、
そのようなパッケージは見つかりませんと返されてしまいます。
ここは ArchWiki を見ればわかるのですが、その後にコアダンプを修正する方法を解説します。
